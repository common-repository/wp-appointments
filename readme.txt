== Description ==

The easy to use free WordPress appointments plugin **WP Appointments** is packed with lots of innovative features that are rarest to be seen elsewhere. For example, both the front end and back end layouts are based totally on AJAX to give a seamless streamlined user experience. The ease of use goes beyond bounds. It can be used for a great variety of websites, such as doctors, spas, saloons, mechanic, personal trainers, lawyers, cleaning services, and so on. The outlook is high interactive and user friendly.

**Features:**

[youtube https://www.youtube.com/watch?v=u_smM0ObkDU]

* Totally AJAX-based Frontend & Backend
* Interactive design with inspiring graphics
* Multipurpose utility 
* Easy & quick installation
* Auto Template Generator included  
* Email Template Builder 
* Managements of multiple locations or branches 
* Management of multiple services
* Management of multiple workers
* Extremely flexible timetable with time slots management 
* Fully device optimize layout
* Translation support

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `wp-appointments.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
